# 🛒 Aspiras Shop

Aspiras Shop is a modern multi-vendor e-commerce web app built using:

- Firebase Authentication
- Firestore Database
- Firebase Hosting
- UPI Payment Integration
- Seller Product Upload System

## 🚀 Features

- User Login / Signup
- Seller Dashboard
- Admin Panel
- Product Image Upload
- UPI Payment (GPay / PhonePe / Paytm)
- Mobile App Style UI
- Firebase Hosting Deployment
